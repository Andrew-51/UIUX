UI vs UX: Understanding the Difference

UI Design

The visual elements users interact with

  • Buttons, menus, typography
  • Color schemes & visual style
  • Layout & composition
  • Responsive design

UX Design

The overall experience & journey

  • User research & personas
  • Information architecture
  • User flows & journey mapping
  • Usability & accessibility

UI Design: DOs and DON'Ts

DO

Use clear visual hierarchy

Guide users through content with size, color, and spacing to indicate importance

DON'T

Create visually flat interfaces

Design with sufficient contrast

Maintain at least 4.5:1 contrast ratio between text and background

Use low contrast elements

Create consistent interactions

Similar elements should respond in predictable ways

Reinvent standard patterns

Accessibility Principles

Perceivable

  • Alt text for images
  • Color contrast (4.5:1 minimum)
  • Text resizing without breaking layout

Operable

  • Keyboard navigation
  • Sufficient time to read content
  • No flashing content

Understandable

  • Clear labels and instructions
  • Consistent navigation
  • Error prevention & suggestions

Robust

  • Compatible with assistive technologies
  • Valid, semantic HTML
  • ARIA when necessary
