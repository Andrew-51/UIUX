What is UI?

User Interface (UI) is everything a user interacts with when using a digital product.

  • Encompasses buttons, forms, typography, colors, layouts, and animations

  • Follows design systems and visual hierarchies
  • Transforms product requirements into accessible interfaces

What is UX?

User Experience (UX) is how users feel when interacting with a product.

  • The entire journey and emotional response of users
  • Encompasses research, testing, structure, and usability
  • Focused on functionality and problem-solving
  • Aims to create meaningful, relevant, and valuable experiences
  • Based on deep understanding of user behaviors and motivations

Don't slice data fields

Do not slice fields when asking for a full name, phone number or date of birth. Sliced fields force the user to make additional taps to move to the next field.

DO

Single fields are easier to complete

DON'T

Multiple fields create unnecessary friction

Accessibility Matters

DO

  • Use sufficient color contrast
  • Support screen readers
  • Provide text alternatives for images
  • Design for keyboard navigation
  • Make touch targets adequately sized

DON'T

  • Rely solely on color to convey information
  • Use tiny text or fonts below 16px
  • Create interfaces that require precision movements
  • Autohide critical navigation
  • Ignore accessibility guidelines

Mobile-First Design

DO

  • Design for thumb zones
  • Prioritize content ruthlessly
  • Use responsive layouts
  • Test on actual devices
  • Consider offline states

DON'T

  • Port desktop layouts directly
  • Use hover-dependent interactions
  • Create tiny touch targets
  • Ignore device capabilities
  • Forget about performance
